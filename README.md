# Asaf Lopez – Digital Resume

This project is a **digital resume** for Asaf Eduardo Lopez Govea, a Senior Front-End Developer. It is built as a modern, responsive web application using **React** and styled with **Tailwind CSS**.

## Features

- Clean, professional layout for presenting resume information
- Responsive design for desktop and mobile
- Sections for summary, technical skills, soft skills, professional experience, early career, and education
- Easily customizable and extendable

## Technologies Used

- [React](https://reactjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)

## Getting Started

1. **Clone the repository:**
   ```sh
   git clone https://github.com/yourusername/your-repo.git
   cd your-repo
   ```

2. **Install dependencies:**
   ```sh
   npm install
   ```

3. **Start the development server:**
   ```sh
   npm start
   ```

4. Open [http://localhost:3000](http://localhost:3000) to view the resume in your browser.

## Project Structure

- `src/App.js` – Main React component rendering the resume
- `src/index.css` – Tailwind CSS and global styles
- `public/` – Static assets and HTML template

## Customization

You can update the resume content by editing the `resumeData` object in `src/App.js`.
